# Simple Encryption Tool

## Description
This is an educational Python project demonstrating a classical substitution cipher.
It allows users to encrypt and decrypt messages using a randomly shuffled key.
The project is intended for learning purposes and to demonstrate basic cryptography concepts.

## Features
- Encrypt messages using substitution cipher
- Decrypt messages
- Simple and well-commented Python code
- Console-based interface
